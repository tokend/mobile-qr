Connect mobile TokenD Wallet to the Developer edition


  1. Install TokenD Wallet to your phone and start TokenD Developer edition on your computer
  2. Install ngrok for your OS

    Why do I need it?

    ngrok allows you to get a public domain for the server running on localhost without the need of real IP and port forwardings.
    However, if you have a real IP with open 8000 port you can reconfigure your environment or just input http://YOUR_IP:8000 on the 4th step.
  3. In the terminal run ngrok http 8000 command
  4. Copy last «‎Forwarding»‎ URL from the appeared window

  5. Paste an URL in the field below and click «‎Get QR code»‎
  6. Scan generated QR code with the mobile TokenD Wallet

Important notice about ngrok

  • Do not turn off ngrok session while using the app
  • Ngrok session may have an expiration period
